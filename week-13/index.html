Challenge

Create a simple signle-channel controller that would allow you to change one parameter in a Processing program by applying a value received from ESP32 via serial communication. Experiment with the parameters you can control with the values.

Try to use a different sensor to generate that value - other than a potentiometer.

Capture images of the build process and a video of the final results. Post with written explanation of the process on your website

  • Starting with a build I had for increasing the diameter of a circle using the knob, I wanted to change the code so when I turned the knob it updated the color fill of the circle. With help from my instructor, we updated the color mode to HSB, created a variable for myColor and mapped the Hue to the knob value.
